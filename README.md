## COMPILAR Y EJECUTAR

Este proyecto utiliza GCC con MinGW para compilar y ejecutar un programa en un entorno Windows.

- **Compilar y ejecutar:** `mingw32-make TP=x & Build\main.exe`
- **Compilar:** `mingw32-make TP=x`
- **Ejecutar:** `Build\main.exe`
